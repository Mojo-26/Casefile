Casefile — Daily Supreme Court Guessing Game

A Wordle-style daily guessing game where you try to identify a famous U.S. Supreme Court case from color-coded clues.

[Play it live →](https://)**

How to play:

A new case is sealed every day at 00:00 UTC — everyone gets the same one. 
Type a case name (or click the search box to browse the full list) and file a guess. Each guess reveals three clues comparing your guess to the sealed case:

| Clue | What it tells you |
| Era | How close your case's decision year is, with an arrow (↑/↓) pointing toward the correct era |
| Type of Law | Whether the legal category matches exactly (green) or is related (amber) |
| Decision| Whether the outcome (affirmed, reversed, etc.) matches |

Stuck? Hit Request a hint to pull an actual passage from the sealed case's majority opinion. 

Once you solve it, you'll see a short summary of the case and can copy your result grid to share.

There's also an unlimited Practice round mode using a random case, which doesn't affect your daily result.

Features

- 30 landmark cases, from Marbury v. Madison (1803) to the 2026 birthright citizenship ruling
- Deterministic daily puzzle — same case for every player, rotates automatically via UTC date
- Progress saved locally in your browser (per device)
- Custom searchable dropdown of the full case list
- Shareable emoji result grid

Running it locally

Just open `index.html` in a browser. That's it.


Case facts (decision year, legal category, outcome, and opinion excerpts) are drawn from published Supreme Court opinions, which are U.S. government works in the public domain. 
Case summaries are written for this project. This project is for entertainment and educational purposes only and is obviously not legal advice.
